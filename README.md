# Nautilus NTC

[![Netlify Status](https://api.netlify.com/api/v1/badges/e1537a8f-c7c8-44b3-b711-df73ad7ca761/deploy-status)](https://app.netlify.com/sites/ntc-web/deploys)

## Requirements

- Node 16

## Quick start

```shell
npm install
```

Run a local server:

```shell
npm start
```

## Development

Run tests:

```shell
npm test
npm run e2e
```

Run checks & formatting:

```shell
npm run lint
npm run format
```

## Production

Build project:

```shell
npm run build
```
